摘要

DDPM, 全名 : Denoising Diffusion Probabilistic Models作為近年來生成模型領域的重大突破,藉由定義一個正向的隱式馬可夫鏈來逐步向數據中添加高斯雜訊，並學習一個反向的去噪馬可夫鏈來從純雜訊中還原出高解析度圖像。本報告完整建構了基於 U-Net 架構的擴散模型，並在經典資料集上進行訓練。實驗結果證實，DDPM 能夠有效捕捉複雜的圖像分佈，生成具備高度多樣性與真實感的圖像樣本。

引言

在生成式 AI 領域中，傳統的生成對抗網路（GAN）雖然能產生高畫質圖像，但常面臨訓練不穩定（Mode Collapse）的問題；而變分自動編碼器（VAE）則容易輸出模糊的結果。

（DDPM）提供了一種全新的機率建模路徑。其基本思想源於非平衡熱力學，藉由將複雜的圖像分佈轉化為簡單的各向同性高斯分佈，再透過深度神經網路精準預估每一步被添加的雜訊，達到漸進式去噪生成的目的。本報告旨在實作此框架，並探討其超參數配置對生成品質的影響。

模型演算法

DDPM 主要由兩個核心過程組成：正向擴散過程（Forward Process）與反向去噪過程（Reverse Process）。

1. 正向擴散過程 (Forward Process)

給定初始真實數據分佈 x₀ ~ q(x)，正向過程在固定步數 T 內，每一步按照預先定義好的方差排程（Variance Schedule）βₜ ∈ (0, 1) 逐步加入高斯雜訊：

q(xₜ | xₜ₋₁) = N(xₜ; √(1 - βₜ)xₜ₋₁, βₜI)

利用重參數化技巧（Reparameterization Trick），我們可以跳過中間步驟，直接從 x₀ 採樣出任意步數 t 的 xₜ：

q(xₜ | x₀) = N(xₜ; √ᾱₜ x₀, (1 - ᾱₜ)I)

其中 αₜ = 1 - βₜ 且 ᾱₜ = ∏ᵗₛ₌₁ αₛ。當 T 足夠大時，x_T 將趨近於標準高斯分佈 N(0, I)。

2. 反向去噪過程 (Reverse Process)

反向過程是擴散模型的精髓。由於真實的反向條件概率 q(xₜ₋₁ | xₜ) 無法直接計算，我們使用一個由參數 θ 控制的神經網路（通常為帶有 Time Embedding 的 U-Net）來近似它：

p_θ(xₜ₋₁ | xₜ) = N(xₜ₋₁; μ_θ(xₜ, t), Σ_θ(xₜ, t))

在實作中，我們通常固定方差 Σ_θ(xₜ, t) = σₜ²I，並讓神經網路去預測正向過程中所加入的雜訊 ε。

3. 演算法虛擬碼實作

以下為訓練與採樣的核心邏輯（PyTorch 風格）：

@torch.no_grad()
def sample(model, n_samples, img_shape, T, beta, alpha, alpha_bar):
    xt = torch.randn(n_samples, *img_shape) # 自標準高斯分佈採樣
    for t in reversed(range(1, T + 1)):
        z = torch.randn_like(xt) if t > 1 else 0
        predicted_noise = model(xt, t)

        xt = mean + sigma_t * z
    return xt

結論與討論

本報告成功重現了經典的 DDPM 框架。實驗表明，模型在去噪的初期（T較大時）主要建立圖像的宏觀結構與輪廓，而在後期（T較小時）則專注於豐富細節與紋理質感的修飾。

研究瓶頸與未來優化方向：

  1. 採樣速度緩慢： 由於生成一張圖片需要完整的序列化走過 1000 步 U-Net 推理，推斷成本極高。後續可嘗試結合 DDIM（跳步非馬可夫採樣）將步數壓縮至 20~50 步，大幅加速推論。
  2. 高維空間計算代價大： 直接在像素空間（Pixel Space）進行擴散會耗費大量顯存。未來可參考穩定擴散（Stable Diffusion）的架構，將數據投影至潛在空間（Latent Space）後再執行擴散過程（即 Latent Diffusion Models）。
